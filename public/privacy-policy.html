Política de Privacidade - WhiteX

Razão Social: Timo Soluções Web e Mobile LTDA
CNPJ: 17.902.232/0001-38
Data de atualização: Janeiro de 2025

1. Informações Gerais

Esta Política de Privacidade descreve como o WhiteX coleta, usa, armazena e protege suas informações pessoais quando você utiliza nossos serviços através do aplicativo móvel ou plataforma web.

2. Dados Coletados

Coletamos apenas os dados essenciais para o funcionamento da plataforma e a segurança do usuário:

  • Informações de cadastro (nome, e-mail, profissão)
  • Dados de saúde inseridos voluntariamente pelo usuário
  • Informações de uso da plataforma para melhorias
  • Dados de verificação profissional (CRM para médicos)

3. Uso dos Dados

Utilizamos suas informações para:

  • Fornecer os serviços da plataforma WhiteX
  • Permitir o compartilhamento seguro com médicos de sua escolha
  • Análise através de inteligência artificial (opcional)
  • Melhorar a experiência do usuário
  • Cumprir obrigações legais

4. Compartilhamento de Dados

Importante: O compartilhamento de dados com médicos se dá única e exclusivamente por iniciativa do paciente, mediante consentimento ativo. Nenhum dado é vendido ou cedido sem o devido consentimento.

5. Segurança e Proteção

Implementamos medidas rigorosas de segurança:

  • Criptografia de dados com padrões internacionais
  • Armazenamento seguro em servidores AWS e Supabase
  • Conformidade com a LGPD (Lei 13.709/18)
  • Backups automáticos e práticas modernas de segurança
  • Acesso restrito e auditoria de sistemas

6. Seus Direitos (LGPD)

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou incorretos
  • Solicitar a exclusão de dados
  • Revogar consentimento a qualquer momento
  • Portabilidade dos dados
  • Informações sobre uso e compartilhamento

7. Inteligência Artificial

Utilizamos IA para auxiliar na organização e análise de dados clínicos. Esta funcionalidade:

  • É opcional e controlada pelo usuário
  • Não substitui diagnóstico médico profissional
  • Processa dados de forma anonimizada quando possível

8. Retenção de Dados

Mantemos seus dados pelo tempo necessário para fornecer nossos serviços ou conforme exigido por lei. Você pode solicitar a exclusão completa a qualquer momento.

9. Atualizações desta Política

Esta política pode ser atualizada periodicamente. Notificaremos sobre mudanças significativas através da plataforma.

10. Contato

Para exercer seus direitos ou esclarecer dúvidas sobre privacidade:

  • E-mail: privacidade@whitex.app.br
  • Através da plataforma WhiteX
Versão para Aplicativo Móvel: Esta política se aplica tanto à versão web quanto ao aplicativo móvel WhiteX. O aplicativo móvel é destinado exclusivamente a pacientes.
